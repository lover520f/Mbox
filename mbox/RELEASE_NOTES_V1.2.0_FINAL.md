## 📋 使用流程

### 首次使用
1. 打开应用
2. 点击"加载配置"
3. 输入配置接口地址
4. 确认加载

### 观看视频
1. 首页浏览推荐内容
2. 点击视频卡片进入详情
3. 选择剧集
4. 点击播放

### 搜索内容
1. 点击底部"搜索"标签
2. 输入关键词
3. 查看搜索结果
4. 点击观看

### 观看直播
1. 点击底部"直播"标签
2. 选择频道分组
3. 选择频道
4. 自动跳转播放器

### 切换站点
1. 点击首页右上角站点名称
2. 从列表中选择站点
3. 自动重新加载内容
