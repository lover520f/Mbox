## 📱 下一步：构建 APK

构建新的 APK 文件需要您手动执行：

### 构建命令

```bash
cd /workspace/mbox
flutter pub get
flutter build apk --split-per-abi --release
```

### 输出文件
- `app-armeabi-v7a-release.apk` (32 位，约 35MB)
- `app-arm64-v8a-release.apk` (64 位，约 40MB)

### 上传 APK

1. 访问 https://github.com/lover520f/Mbox/releases/tag/v1.1.0
2. 点击 "Edit"
3. 上传 APK 文件到 "Attach binaries" 区域
4. 点击 "Save changes"
